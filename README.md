# How to Tar Zip your files

In the directory containg the files you want to tar right click and open a terminal.
```bash
tar cvf **StudentNumber**.tar {**question1.py**,**question2.py**}
```
**PRESS ENTER**
```bash
bzip2 **StudentNumber**.tar
```
**PRESS ENTER**
In your directory you should now see **StudentNumber**.tar.bz2
